Mengenal Frontend Development dan Perannya dalam Website

20 Desember 2025 Frontend

Frontend development adalah bagian dari pengembangan website yang berfokus pada tampilan dan interaksi pengguna. Semua hal yang dapat dilihat dan digunakan langsung oleh pengguna, seperti warna, tombol, teks, dan animasi, merupakan tanggung jawab frontend developer.

Dalam pengembangan frontend, terdapat tiga teknologi utama yang paling sering digunakan, yaitu HTML, CSS, dan JavaScript. HTML berfungsi untuk membangun struktur halaman, CSS digunakan untuk mengatur tampilan dan desain, sedangkan JavaScript membuat website menjadi interaktif.

Seorang frontend developer juga harus memperhatikan user experience (UX) dan user interface (UI). Website yang baik bukan hanya terlihat menarik, tetapi juga mudah digunakan dan responsif di berbagai perangkat seperti laptop, tablet, dan smartphone.

Website dikembangkan menggunakan HTML, CSS, dan JavaScript dengan menerapkan desain responsif serta dark mode.
