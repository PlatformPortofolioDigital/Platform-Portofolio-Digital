Website Portfolio

Project pengembangan website portfolio statis sebagai media dokumentasi, pembelajaran, dan presentasi karya Kelompok 2.

Deskripsi Project

Website portfolio ini dibuat untuk menampilkan profil kelompok, daftar project, serta dokumentasi hasil pembelajaran selama perkuliahan. Website dikembangkan menggunakan teknologi web dasar dengan struktur yang sederhana, rapi, dan mudah dikembangkan.

Tujuan Pembuatan

  • Menjadi media portfolio resmi Kelompok 2
  • Melatih penerapan HTML, CSS, dan JavaScript
  • Menyediakan dokumentasi project secara online

Teknologi yang Digunakan

  • HTML5
  • CSS3
  • JavaScript

Kontribusi Tim

  • Perancangan struktur halaman
  • Implementasi desain UI
  • Pengujian dan penyempurnaan tampilan
