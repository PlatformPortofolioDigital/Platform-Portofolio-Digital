Sistem Informasi Sederhana

Aplikasi CRUD sederhana yang dikembangkan untuk memahami konsep dasar sistem informasi.

Deskripsi Project

Project ini bertujuan untuk melatih pemahaman alur data, operasi CRUD (Create, Read, Update, Delete), serta struktur sistem informasi sederhana berbasis web.

Tujuan Pembuatan

  • Melatih konsep dasar sistem informasi
  • Memahami alur data CRUD
  • Mengimplementasikan logika sederhana berbasis web

Teknologi yang Digunakan

  • HTML
  • CSS
  • JavaScript
