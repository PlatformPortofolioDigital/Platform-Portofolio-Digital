Deskripsi Project

Project ini dikembangkan untuk melatih pemahaman alur data, operasi CRUD (Create, Read, Update, Delete), serta penerapan logika sistem informasi sederhana.

Tujuan Pembuatan

  • Memahami konsep dasar sistem informasi
  • Mengimplementasikan operasi CRUD
  • Melatih logika pemrograman berbasis web

Fitur Utama

  • Tampilan data terstruktur
