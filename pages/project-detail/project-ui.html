UI Design Prototype

Project desain antarmuka aplikasi dengan pendekatan UI/UX yang sederhana dan modern.

Deskripsi Project

Prototype ini dibuat untuk melatih kemampuan desain antarmuka pengguna yang fokus pada kejelasan visual, konsistensi layout, dan kenyamanan pengguna dalam berinteraksi dengan aplikasi.

Tujuan Pembuatan

  • Memahami prinsip dasar UI/UX
  • Melatih perancangan layout yang konsisten
  • Meningkatkan pengalaman pengguna (user experience)

Tools & Konsep

  • UI/UX Principles
  • Wireframing
  • Design Consistency
