Platform Portofolio Digital adalah sebuah **website portofolio berbasis web** yang dikembangkan sebagai **proyek tugas kelompok**.  
Website ini berfungsi untuk menampilkan **identitas tim, portofolio project, artikel/blog, serta halaman kontak** dengan tampilan modern, interaktif, dan responsif.

Project ini dirancang untuk melatih kemampuan **Frontend Web Development**, sekaligus membiasakan kerja tim dan kolaborasi menggunakan **Git & GitHub**.


##  Tujuan Project

- Menyajikan portofolio digital dalam bentuk website yang modern dan profesional
- Melatih kerja tim serta kolaborasi berbasis GitHub
- Mengimplementasikan HTML, CSS, dan JavaScript sesuai standar pengembangan web
- Menerapkan desain responsif yang dapat diakses di berbagai perangkat
- Menghasilkan website portofolio yang layak dipublikasikan

##  Fitur Utama
-  **Welcome Screen**
-  **Home / Landing Page** dengan hero section interaktif  
-  **About** (Profil tim & anggota)
-  **Detail About** (Profil individu tiap anggota)
-  **Projects / Portofolio**
-  **Detail Project**
-  **Blog / Artikel**
-  **Detail Blog**
-  **Contact / Get In Touch**
-  **Dark Mode & Light Mode**
-  **Typing Text Animation**
-  **Smooth Navigation & Page Transition**
-  **Responsive Design (Mobile Friendly)**
- **Validasi Form menggunakan JavaScript**
